# s-note

基于 Markdown 的轻量文档仓库，使用 GitHub Pages 托管。

## 本地约定
- 文档放在 `docs/`
- 文档取名以`文档序号_文档名`的方式，其中文档序号是`日期+序号`(日期为yyyyMMdd，序号为两位数从00开始)
- 图片放在 `docs/assets/images/`（保证 GitHub Pages 发布时可访问）
- 文档专属图片，其命名方式为`文档序号_图片名`，方便以后对文档删除时的图片管理
- 非文档专属图片，统一前缀为`C_`，后跟图片名，图片名应使用英文尽量描述图片功能，不要包含与文档序号相同的内容，便于筛选
- 全部使用**相对路径**引用，方便后续迁移到任意平台
