# 如何模拟磁盘满
在日常开发过程中，可能会遇到磁盘空间不足导致程序运行异常的场景，用大文件填充磁盘是一种方法，但比较影响日常使用，构建一个小磁盘更容易触发。
## Windows
在Windows平台，有一些第三方工具，但个人更倾向于使用Windows平台的默认能力，创建小容量虚拟磁盘，而且方便进行自动化。
### 创建小容量虚拟磁盘
#### 图形界面操作
##### 1.打开磁盘管理
按Win + R输入diskmgmt.msc
![打开磁盘管理](../../assets/images/2026041000_01.jpg)
![磁盘管理页面](../../assets/images/2026041000_02.jpg)
##### 2.创建或附加VHD
![创建或附加VHD](../../assets/images/2026041000_03.jpg)
###### 2.1 创建VHD
"**操作**" -> "**创建VDH**"
![创建VHD](../../assets/images/2026041000_04.jpg)
在磁盘上右击，选择**初始化磁盘**
![初始化磁盘](../../assets/images/2026041000_06.jpg)
![初始化磁盘2](../../assets/images/2026041000_07.jpg)
###### 2.2 附加VHD
"**操作**" -> "**附加VDH**"
![附加VHD](../../assets/images/2026041000_05.jpg)
#### 3.新建简单卷
在未分配空间上右击，选择**新建简单卷**
![新建简单卷](../../assets/images/2026041000_08.jpg)
分配驱动器号
![分配驱动器号](../../assets/images/2026041000_09.jpg)
检查结果
![检查结果](../../assets/images/2026041000_10.jpg)
#### 4.删除卷
删除卷，也可忽略这一步直接执行**分离VHD**
![删除卷](../../assets/images/2026041000_11.jpg)
#### 5.分离VHD
在磁盘上右击，选择"**分离VHD**"
![分离VHD](../../assets/images/2026041000_12.jpg)
