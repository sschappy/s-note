# 如何设置文件夹权限
日常开发总会遇到文件夹权限不足的场景，对于Linux平台，基本上都是使用现有的指令，但对于Windows平台，大多时候还是GUI操作。
## Windows
### GUI手动设置
##### 1.创建文件夹
##### 2.设置权限
1. 右击文件夹 → 属性 → 安全 → 高级
2. 点击"禁用继承" → 选择"从此对象中删除所有已继承的权限"
3. 点击"添加" → 选择主体 → 输入"SYSTEM" → 确定
4. 基本权限选择"完全控制" → 确定
5. 再次点击"添加" → 选择主体 → 输入"Administrators" → 确定
6. 基本权限选择"完全控制" → 确定
7. 再次点击"添加" → 选择主体 → 输入当前用户名 → 确定
8. 基本权限只勾选：
   - ☑ 读取和执行
   - ☑ 列出文件夹内容
   - ☑ 读取
   - ☐ 写入（不要勾选！）
   - ☐ 修改（不要勾选！）
   - ☐ 完全控制（不要勾选！）
9.  点击确定 → 确定 → 确定
##### 3.验证设置
尝试在文件夹中：
- 创建新文件 ❌ 失败
- 创建子文件夹 ❌ 失败
- 修改现有文件 ❌ 失败
- 读取文件 ✔ 成功
- 列出内容 ✔ 成功
